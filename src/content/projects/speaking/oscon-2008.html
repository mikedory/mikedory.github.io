This session will feature an introduction to physical computing and interfacing with microcontrollers, as well as the basics of Arduino (hardware and software) and using it with the Processing programming environment. --- ## Making Things Blink Arduino is an open-source electronics prototyping platform based on flexible, easy-to-use hardware and software. It's intended for artists, designers, hobbyists, and anyone interested in creating interactive objects or environments. In this tutorial, participants will learn how to create devices for sensing and communicating with the physical world using the Arduino platform. This session will feature an introduction to physical computing and interfacing with microcontrollers, as well as the basics of Arduino (hardware and software) and using it with the Processing programming environment. ### Featured in: - [MAKE Blog](http://blog.makezine.com/2008/08/05/making-things-talkblink-a/)
